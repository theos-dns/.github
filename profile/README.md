## Theos DNS

### Ports:
#### IR server:
- `53`   => DNS server
- `853`  => DoT server
- `3443` => DoH server
- `81`   => get public ip address
- `82`   => auth api (to register user and new ip)
- `85`   => COAP service

#### EU server:
- `83` => xray-ui panel (marzban)
